# 导入litemall数据库 - 快速指南

## 方法1: 使用脚本（推荐）

```bash
cd D:\Code\litemall-master
import-mysql.bat
```

脚本会提示输入密码。

---

## 方法2: 手动导入

### 步骤1: 打开MySQL命令行

```bash
mysql -u root -p
```

输入你的MySQL root密码。

### 步骤2: 复制粘贴以下SQL

```sql
-- 1. 创建数据库
SOURCE D:\Code\litemall-master\litemall-db\sql\litemall_schema.sql;

-- 2. 创建表
USE litemall;
SOURCE D:\Code\litemall-master\litemall-db\sql\litemall_table.sql;

-- 3. 导入数据
SOURCE D:\Code\litemall-master\litemall-db\sql\litemall_data.sql;
```

---

## 方法3: 使用MySQL Workbench

1. 打开 MySQL Workbench
2. 连接到本地MySQL服务器
3. 依次打开并执行以下文件:
   - `litemall-db/sql/litemall_schema.sql`
   - `litemall-db/sql/litemall_table.sql`
   - `litemall-db/sql/litemall_data.sql`

---

## 验证数据库

```sql
USE litemall;
SHOW TABLES;

-- 应该看到30+张表

-- 检查管理员用户
SELECT * FROM litemall_admin WHERE username = 'admin123';
```

---

## 常见问题

### Q: 忘记MySQL密码？
A:
- Windows: 重置MySQL服务
- 或重新安装MySQL

### Q: 执行SQL出错？
A:
1. 确保MySQL服务正在运行
2. 检查文件路径是否正确
3. 查看具体错误信息

---

## 导入完成后

启动后端服务：

```bash
cd D:\Code\litemall-master
start-backend.bat
```

后端启动成功后，管理后台就能正常登录了！
