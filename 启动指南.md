## 启动前准备

### 1. 安装MySQL

为了快速安装MySQL，我们提供了安装向导脚本：

**🚀 快速安装（推荐）:**
```bash
cd D:\Code\litemall-master
install-mysql.bat
```

这个脚本会提供4个选项：
1. **MySQL Installer** - 图形界面安装，最简单
2. **ZIP版下载** - 下载国内镜像，解压即用
3. **Docker方式** - 最快，需要已安装Docker
4. **跳过** - 稍后手动安装

**详细指南**: 请查看 `D:\DevTools\MySQL安装指南.md`

### 2. 导入数据库

根据你选择的安装方式，使用对应的导入脚本：

**方式A: 本地MySQL安装后导入**
```bash
cd D:\Code\litemall-master
import-mysql.bat
```

**方式B: Docker容器导入**
```bash
cd D:\Code\litemall-master
import-mysql-docker.bat
```

手动导入（如果脚本失败）:
```bash
cd D:\Code\litemall-master\litemall-db\sql

# 1. 创建数据库结构
mysql -u root < litemall_schema.sql

# 2. 创建表
mysql -u root litemall < litemall_table.sql

# 3. 导入初始数据
mysql -u root litemall < litemall_data.sql
```

或在MySQL Workbench中：
1. 打开 MySQL Workbench
2. 连接到本地MySQL服务器
3. 依次执行 litemall_db/sql/ 目录下的三个SQL文件

### 3. 配置数据库连接

编辑以下文件，修改数据库连接信息：
- `litemall-wx-api/src/main/resources/application.yml`
- `litemall-admin-api/src/main/resources/application.yml`

修改数据库配置：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/litemall?useUnicode=true&characterEncoding=UTF-8&serverTimezone=Asia/Shanghai
    username: root
    password: 你的MySQL密码
```

## 启动后端服务

### 方式1: 使用JAR启动（推荐）
```bash
java -Dfile.encoding=UTF-8 -jar D:\Code\litemall-master\litemall-all\target\litemall-all-0.1.0-exec.jar
```

### 方式2: 使用Maven启动
```bash
cd D:\Code\litemall-master
mvn spring-boot:run -pl litemall-all
```

后端服务启动后，将监听以下端口：
- 管理后台API: http://localhost:8080/admin
- 微信小程序API: http://localhost:8080/wx

## 启动前端

### 管理后台 (litemall-admin)
```bash
cd D:\Code\litemall-master\litemall-admin
npm install
npm run dev
```
访问: http://localhost:9527

### 微信小程序 (litemall-wx)

## 常见问题

### 环境变量未生效
关闭当前命令行窗口，重新打开后执行 `java -version` 和 `mvn -version` 验证

### 端口冲突
如果端口被占用，可以修改 application.yml 中的端口配置

### 数据库连接失败
检查MySQL服务是否启动，用户名密码是否正确
