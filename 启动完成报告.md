## ⚠️ 重要提醒

### 数据库必须导入！

**后端无法启动的原因**：
- MySQL数据库未导入
- litemall数据库不存在

**必须执行**：
```bash
cd D:\Code\litemall-master
import-mysql.bat
```

### MySQL密码问题

如果你的MySQL设置了密码：

1. **编辑配置文件**:
   - `litemall-wx-api/src/main/resources/application.yml`
   - `litemall-admin-api/src/main/resources/application.yml`

2. **修改密码**:
```yaml
spring:
  datasource:
    password: 你的MySQL密码
```
