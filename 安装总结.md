## 📋 下一步操作

### 🔴 必需：安装MySQL数据库

我们提供了快速安装脚本：

```bash
cd D:\Code\litemall-master
install-mysql.bat
```

**4个安装选项：**

1. **MySQL Installer** (推荐新手)
   - 图形界面安装向导
   - 自动配置，最简单
   - 需要下载约450MB

2. **ZIP版下载** (推荐开发)
   - 使用国内镜像快速下载
   - 解压即用，无需安装
   - 需要手动初始化

3. **Docker** (最快)
   - 需要先安装Docker Desktop
   - 一条命令启动MySQL
   - 方便管理和删除

4. **手动安装**
   - 稍后自己安装
   - 参考 `D:\DevTools\MySQL安装指南.md`

### 📦 导入数据库

根据你的MySQL安装方式选择：

**本地MySQL:**
```bash
cd D:\Code\litemall-master
import-mysql.bat
```

**Docker MySQL:**
```bash
cd D:\Code\litemall-master
import-mysql-docker.bat
```

## ⚠️ 注意事项

1. **首次启动前必须安装MySQL并导入数据库**
2. 如果环境变量不生效，请重新打开命令行窗口
3. 管理后台默认端口9527，Vue移动端端口6255
4. 后端API端口8080
5. 确保MySQL的3306端口未被占用

## 🐛 常见问题

### 环境变量未生效
关闭当前命令行窗口，重新打开后验证：
```bash
java -version
mvn -version
```

### MySQL连接失败
1. 检查MySQL服务是否启动: `net start MySQL80`
2. 检查数据库配置文件中的用户名密码
3. 确认3306端口未被占用

### 端口冲突
修改 `application.yml` 中的端口配置

### 前端npm install失败
使用淘宝镜像：
```bash
npm install --registry=https://registry.npmmirror.com
```
