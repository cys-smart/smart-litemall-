# litemall 初始化脚本系统

## ✅ 已创建的脚本

### 1. 一键初始化.bat
**完整初始化环境** - 首次使用必须运行

**功能**：
- 配置 JAVA_HOME 环境变量
- 配置 MAVEN_HOME 环境变量
- 配置 PATH 环境变量
- 启动 MySQL80 服务
- 导入 litemall 数据库
- 创建 litemall 数据库用户

**使用方法**：右键 → 以管理员身份运行

---

### 2. 启动系统.bat
**启动项目服务** - 内置环境变量，不依赖系统配置

**功能**：
- 可选择性启动后端/前端
- 自动使用完整路径调用Java
- 多窗口启动，互不干扰

**选项**：
- [1] 仅后端
- [2] 仅管理后台
- [3] 仅Vue移动端
- [4] 全部启动
- [5] 后端+管理后台

---

### 3. 系统诊断.bat
**检查系统状态** - 排查问题必备

**检查项**：
- Java环境
- Maven环境
- 环境变量配置
- MySQL服务状态
- 数据库状态
- 项目文件完整性

---

### 4. 配置环境变量.bat
**单独配置环境变量** - 可选

**功能**：
- 设置用户环境变量
- 无需重启电脑

---

## 🚀 使用流程

### 首次安装
```
1. 右键"一键初始化.bat" → 以管理员身份运行
   ↓
2. 关闭命令行，重新打开
   ↓
3. 双击"启动系统.bat"
```

### 日常启动
```
直接双击"启动系统.bat"
```

### 遇到问题
```
双击"系统诊断.bat"查看状态
```

---

## 📝 注意事项

1. **管理员权限**：`一键初始化.bat` 和 `配置环境变量.bat` 需要管理员权限
2. **环境变量**：配置后需要重新打开命令行窗口才能生效
3. **MySQL服务**：如果MySQL服务名不是 MySQL80，需要手动启动
4. **数据库密码**：如果MySQL有root密码，初始化脚本会提示输入
