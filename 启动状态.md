## ⚠️ 重要提醒

### 数据库配置

**后端可能无法连接数据库！**

需要导入litemall数据库，请运行：

```bash
cd D:\Code\litemall-master

# 方式1: 使用脚本（推荐）
import-mysql.bat

# 方式2: 手动导入
cd litemall-db\sql
mysql -u root -p < litemall_schema.sql
mysql -u root -p litemall < litemall_table.sql
mysql -u root -p litemall < litemall_data.sql
```

**如果MySQL有密码:**
编辑以下文件修改数据库密码：
- `litemall-wx-api/src/main/resources/application.yml`
- `litemall-admin-api/src/main/resources/application.yml`

```yaml
spring:
  datasource:
    password: 你的MySQL密码
```

## 🐛 常见问题

### Q: 后端启动失败？
A:
1. 检查MySQL服务是否运行: `net start MySQL80`
2. 检查数据库是否已导入
3. 查看后端日志错误信息

### Q: 前端npm install太慢？
A:
- 正在使用的淘宝镜像，请耐心等待
- 首次安装约需2-5分钟
- 可以查看下载进度

### Q: 端口被占用？
A:
```bash
# 查找占用端口的进程
netstat -ano | findstr "端口号"

# 结束进程
taskkill /F /PID 进程ID
```

### Q: 管理后台页面空白？
A:
- npm install可能未完成
- 等待完成后刷新页面
- 检查浏览器控制台错误
